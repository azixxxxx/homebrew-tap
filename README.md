# Homebrew Tap for Glance

Custom macOS status bar with liquid glass UI, native Spaces support, and configurable widgets.

## Install

```bash
brew tap azixxxxx/tap
brew install --cask glance
```

## Update

```bash
brew upgrade --cask glance
```
